# ClipShare - Anonymous Real-time Clipboard

Share any file or text instantly with a 6-digit code + password. No login, no phone number, no email. Files auto-delete after 1 hour.

## ✨ Features
- 🔐 Code + password protected sessions
- 📱 Works on any device (mobile + desktop)
- 🎯 Real-time text sync (shared notepad)
- 📁 Share any file type (images, videos, docs, code, zip)
- ⏱️ 1-hour auto-expiry (everything deleted)
- 🔄 Reusable same session for multiple files
- QR code for easy sharing
- No user accounts, no personal data
